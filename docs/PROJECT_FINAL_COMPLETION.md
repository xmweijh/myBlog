## 🚀 快速开始

### 1. 启动后端服务

```bash
cd backend
npm install
npm run dev
```

后端运行在 `http://localhost:3001`

### 2. 启动前端开发服务器

```bash
cd frontend
npm install
npm run dev
```

前端运行在 `http://localhost:5173`

### 3. 测试账号

```
邮箱: admin@myblog.com
密码: admin123456
角色: ADMIN
```

### 4. 创建新账号

访问 `http://localhost:5173/register` 创建新账号

### 5. 访问管理后台

使用管理员账号登录后，点击导航栏的"管理后台"按钮
