## 🚀 使用示例

### 基础使用

```typescript
import AdminPage from '@/pages/AdminPage'

// 在路由中使用
<Route path="/admin" element={<AdminPage />} />
```

### 访问管理后台

```typescript
// 从导航栏访问
<Link to="/admin" className="...">
  管理后台
</Link>

// 或直接访问
http://localhost:5173/admin
```

### 权限检查

```typescript
// 只有管理员可以访问
if (user.role !== 'ADMIN') {
  navigate('/')
}
```

## 💡 最佳实践

### 1. 权限检查

```typescript
// 检查认证状态
useEffect(() => {
  if (!isAuthenticated || !user) {
    navigate('/login')
    return
  }

  // 检查管理员权限
  if (user.role !== 'ADMIN') {
    navigate('/')
    return
  }
}, [isAuthenticated, user, navigate])
```

### 2. 统计数据加载

```typescript
// 模拟加载统计数据
useEffect(() => {
  setTimeout(() => {
    setStats({
      totalArticles: 4,
      totalCategories: 5,
      // ...
    })
    setIsLoading(false)
  }, 500)
}, [])
```

### 3. 快速操作链接

